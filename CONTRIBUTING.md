# Contributing

Thanks for contributing to ai-decision-router!

## Setup
1. Fork and clone the repository.
2. Install dependencies:
   ```bash
   make setup
   ```
3. Run checks before opening a PR:
   ```bash
   make fmt
   make lint
   make test
   ```

## Pull request guidelines
- Keep changes focused and well-tested.
- Add/update docs for behavior changes.
- Include tests for new logic (especially policies and classifier behavior).

## Reporting issues
Please include reproduction steps, expected behavior, actual behavior, and environment details.
